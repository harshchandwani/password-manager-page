Tech Stack

Frontend:

  • NextJs: Pages
  • TailwindCSS: Initial Design
  • Lucide React: Icons
  • ShadCn: Designing

Backend:

  • Nodejs: Build
  • Express
  • Redis: Queue for Email

Others:

  • Postgres: Database
  • BcryptJs: Password Encryption
  • JsonwebToken: Tokens
  • NodeMailer: Email
  • Express Rate Limit: Rate Limiting

To Use

Clone the project

git clone https://github.com/harshchandwani/password-manager-backend
git clone https://github.com/harshchandwani/password-manager-frontend

Go to the project directory

cd password-manager-backend

Install dependencies in Backend

npm install

Start the server in Frontend

npm run dev

Go to project directory and in Backend

cd ..
cd password-manager-backend

Install dependencies in Backend

npm install

Start the server in Frontend

npm run start
